# Mortal! — Match parser

Short utility to extract and format match blocks from `match.txt`.

Usage
- Run: `python main.py` and follow the prompt to enter a match id (example `N215`).

Quick check
- `main.py` parses blocks starting with `#N...`, transliterates/normalizes names, and formats output.
- I ran a quick static check: no syntax errors found.
